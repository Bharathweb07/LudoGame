LUDO

A Classic Board Game Experience Reimagined

General Information

Platform

Android (Version 7.0 & Higher)

Package Name

com.example.ludo

Current Version

1.0.0 (Initial Release)

Key Features

  • Multiplayer Support: Play with 2, 3, or 4 players locally on a single device.
  • Intelligent AI: Challenging CPU opponents with customizable toggles for every player slot.
  • Authentic Mechanics: Includes standard rules like rolling a 6 to start, capturing opponent tokens, and bonus turns.
  • Optimized UI: Adaptive board layout that adjusts based on the number of active players.
  • Offline Play: Full functionality without requiring an internet connection or data usage.

Technical Specifications

Java Android SDK Custom Canvas API Adaptive Icons Vector Graphics

The app utilizes a custom Game Engine architecture to separate logic from rendering, ensuring smooth token animations and high-performance dice physics.

Visual Identity

Classic color-coded gameplay with high-contrast themes:

Yellow (Player 1)
Green (Player 2)
Red (Player 3)
Blue (Player 4)
